Viewing CSV Files IV

Agility Kata “Viewing CSV Files IV”

Enhance the CSV viewer so that when opening the application again on a CSV file it has already processed (partly or fully)[1], processing does not start all over. Instead processing should continue from what position in the CSV file it had reached last time when the application was exited.

[1] …and the CSV file has not changed since then
